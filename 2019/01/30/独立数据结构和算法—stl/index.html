本篇重点-算法

sort排序

第一种情况

#include <bits/stdc++.h>
using namespace std;
int main()
{
int a[]={1,5,3,6,4};
sort(a,a+sizeof(a)/sizeof(int));
for (int i = 0; i < sizeof(a)/sizeof(int); ++i)
{
    cout<<a[i]<<" ";
}
return 0;
}

这是最简单的一种情况，将一个数组从大到小排序
这当中不一定是对全部内容进行排序，可以对部分连续数组进行排序，规则为左闭右开。

第二种情况

#include <bits/stdc++.h>
using namespace std;
int main()
{
int a[]={1,5,3,6,4};
sort(a,a+sizeof(a)/sizeof(int),greater<int>());
for (int i = 0; i < sizeof(a)/sizeof(int); ++i)
{
    cout<<a[i]<<" ";
}
return 0;
}

这一种用了greater,将数组给从大到小输出

第三种情况

#include <bits/stdc++.h>
using namespace std;
struct fun
{
    bool operator() (const int &a,const int &b){
       return a%10>b%10;
}    
};
int main()
{
int a[]={19,51,32,6,4};
sort(a,a+sizeof(a)/sizeof(int),fun());
for (int i = 0; i < sizeof(a)/sizeof(int); ++i)
{
    cout<<a[i]<<" ";
}
return 0;
}

这当中采用了自定义规则排序，主要是要注意自定义规则的写法。
